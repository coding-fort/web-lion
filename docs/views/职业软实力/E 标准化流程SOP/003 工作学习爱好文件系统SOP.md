# 工作 - 学习 - 爱好文件系统 SOP

在 macOS 上搭建工作 - 学习 - 爱好的文件系统，可依托访达（Finder）的层级文件夹为核心，搭配标签、智能文件夹等系统功能提升管理效率，同时结合命名规范和备份策略保障文件安全与检索便捷性。以下是具体的搭建方案：

## 一、基础文件夹结构（存于“文稿”目录）

macOS 的“文稿”目录是个人文件管理的核心位置，以此为根目录搭建三级文件夹结构，层级清晰且便于 iCloud 同步。按下`Shift+Command+N`可快速新建文件夹，具体结构如下：

```
文稿
├─ 01_工作
│  ├─ 01_当前项目
│  │  ├─ 项目A（如202511_深圳宝安项目）
│  │  │  ├─ 00_项目资料（需求文档、对接人信息等）
│  │  │  ├─ 01_过程文件（会议纪要、草稿版方案等）
│  │  │  └─ 02_最终输出（定稿文件、交付成果等）
│  │  └─ 项目B
│  ├─ 02_日常事务
│  │  ├─ 考勤薪资
│  │  ├─ 周报月报
│  │  └─ 行政通知
│  └─ 03_资源储备
│     ├─ 行业报告
│     ├─ 工具安装包
│     └─ 模板库（PPT、Excel模板等）
├─ 02_学习
│  ├─ 01_当前课程（如2025秋_数据分析课程）
│  │  ├─ 00_课件资料
│  │  ├─ 01_作业练习
│  │  └─ 02_笔记总结
│  ├─ 02_考证备考（如PMP备考）
│  │  ├─ 真题试卷
│  │  ├─ 复习提纲
│  │  └─ 错题整理
│  └─ 03_技能积累
│     ├─ 编程学习（如Python代码）
│     ├─ 语言学习（如英语听力素材）
│     └─ 阅读笔记（书籍摘要、感悟等）
└─ 03_爱好
    ├─ 摄影
    │  ├─ 原图备份
    │  ├─ 后期处理
    │  └─ 作品归档
    ├─ 音乐
    │  ├─ 乐谱资料
    │  └─ 录制作品
    └─ 手工
        ├─ 教程文档
        └─ 成品照片
```

## 二、标签体系搭配（快速区分优先级）

利用 macOS 的标签功能实现跨文件夹分类，比如区分紧急程度或文件类型，右键文件即可添加标签，具体设置如下：
|标签颜色|标签名称|适用场景|
| ---- | ---- | ---- |
|红色|紧急重要|工作里待交付的紧急项目文件、学习中次日要交的作业|
|橙色|待处理|非紧急的日常工作任务、需慢慢整理的学习资料|
|绿色|已完成|归档的完成项目、批改后的作业、定稿的爱好作品|
|蓝色|参考类|工作行业报告、学习拓展资料、爱好相关教程|
|紫色|私密类|涉及薪资的文件、个人学习复盘、未公开的爱好作品|

## 三、智能文件夹设置（自动化归类）

智能文件夹能自动聚合符合条件的文件，无需手动移动，适合高频查看的文件类型，以下是实用配置：

1.  **工作紧急文件**：打开访达按`Command+F`，添加条件“标签 是 紧急重要”+“上次修改日期 在过去 3 天内”，点击保存命名为“工作紧急待处理”，勾选添加到侧边栏。
2.  **学习作业汇总**：搜索条件设为“文件夹 包含 作业练习”+“种类 是 文稿”，保存为“近期作业”，快速查看所有待完成和已完成的作业。
3.  **爱好作品合集**：条件设为“文件夹 包含 作品归档”+“种类 是 图像”，保存为“爱好作品库”，自动汇总各类爱好的成品文件。

## 四、进阶优化技巧

1.  **命名规范**：统一采用`日期_内容_版本`格式，例如`20251111_宝安项目方案_v2`，搭配 Spotlight 搜索（`Command+空格`）输入关键词即可快速定位文件。
2.  **同步与备份**：在“系统设置 - Apple ID - iCloud”中勾选“iCloud 云盘”，同步“工作”和“学习”的核心文件夹，方便跨设备查看；爱好的大文件（如摄影原图）可存于外置硬盘，通过 Time Machine 定期备份。
3.  **快捷键与快速访问**：将“01*当前项目”“01*当前课程”等常用文件夹拖拽到访达侧边栏；记住`Command+Option+V`移动文件、`Command+D`复制文件，提升操作效率。
4.  **定期清理**：每月整理一次文件夹，删除无用的草稿文件，将过期项目、完成课程归档到对应年份的备份文件夹，避免目录杂乱。
