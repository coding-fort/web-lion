# 《如何渡过每天的 24 小时》

## 时间管理三大原则

### 1.时间账户

- 生存时间：满足基本生存需求的时间
- 工作时间：用于完成工作任务的时间
- 主权时间：完全由自己掌控、可支配用于个人兴趣、成长、休闲等的时间

### 2.十五分钟法则：任何超过十五分钟的时间都值得认真投资。

### 3.时间（注意力）复利：持续在同一领域或某某事情上投入碎片化时间，会有惊人的累积结果。
